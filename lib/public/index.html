Route Method Description Example

Plants Resource

 
/plants
 GET Gets a list of all plants. List Plants 
/plants/1
 GET Gets a specific plant. List Plant 
/plants/
 POST Adds a plant. curl -d '{"name": "Succulent", "scientific_name": "Succulent"}' -H 'Content-Type: application/json' 'https://module3-hand-of-resources.herokuapp.com/plants' 
/plants/2
 PUT Updates a plant. curl -X PUT -H "Content-Type: application/json" -d '{"name":"Monstera"}' https://module3-hand-of-resources.herokuapp.com/plants/2 
/plants/2
 DELETE Deletes a plant. curl -X DELETE https://module3-hand-of-resources.herokuapp.com/plants/2 -H "Accept: application/json"
